# ⚠️ Проект закрыт

Этот форк больше не поддерживается, так как **не совместим с хостингом** и не работает с Gemini API так, как задумывалось.

---

## 🚫 Причина закрытия

- ❌ Netlify не смог корректно обработать сборку Next.js с нестандартной структурой.
- ❌ Встроенный интерфейс `NextChat` упорно переключался на `gpt-4o-mini`, игнорируя выбор `Gemini`.
- ❌ API-ключи либо не передавались в окружение, либо модель выдавала ошибку `404`.
- ❌ После всех исправлений интерфейс либо падал, либо отвечал ошибкой `429`, что свидетельствует о проблемах с квотой.
- ❌ Проект изначально не рассчитан на бесплатный хостинг с ограниченными ресурсами.
